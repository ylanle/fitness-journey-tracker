# Gym Progress Tracker

Mini Project 2 is a single-page fitness dashboard built with plain HTML, CSS, and vanilla JavaScript.

## Features

- Nutrition tracker for date, meal name, meal type, calories, protein, carbs, and fat.
- Workout tracker for date, exercise, weight, sets, reps, workout volume, and personal records.
- Body measurement tracker for body weight, optional body fat percentage, latest weight, and weight change.
- Dashboard section with daily totals, latest body weight, total workout volume, PR highlights, and a simple SVG chart.
- Settings for units and daily calorie/macro targets.
- Edit and delete buttons for entries.
- Clear-all reset button.
- Data persistence with `localStorage`.

## Tech Requirements

- Plain HTML, CSS, and JavaScript only.
- No frameworks, no libraries, no CDN, and no API.
- Runs on GitHub Pages by opening `index.html`.

## Files

- `index.html` contains the semantic page structure.
- `style.css` contains the responsive dashboard styling.
- `script.js` contains the JavaScript state, validation, localStorage, calculations, and DOM updates.
